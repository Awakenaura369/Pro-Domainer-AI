# 🚀 AI Domain Hunter - Pro Edition
An intelligent domain name generator powered by **Groq AI (Llama 3)** and **Streamlit**. Designed for domain investors to find high-value, brandable, and liquid .com domains in seconds.

## 🌟 Features
- **Professional Logic:** Uses specialized AI prompts to think like a veteran domain flipper.
- **Real-time Valuation:** Provides estimated resale values based on market trends.
- **Instant Availability Check:** One-click redirection to Namecheap for immediate registration.
- **Modern UI:** Clean and responsive interface built with Streamlit.
- **Fast Performance:** Leveraging Groq's high-speed inference for near-instant results.

## 🛠️ Tech Stack
- **Frontend:** [Streamlit](https://streamlit.io/)
- **LLM Engine:** [Groq API](https://groq.com/) (Llama 3 8B)
- **Data Handling:** [Pandas](https://pandas.pydata.org/)
- **Language:** Python 3.9+
